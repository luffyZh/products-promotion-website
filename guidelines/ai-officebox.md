# 模块一：首屏（Hero Section） - 核心价值主张

【设计目标】 第一时间抓住访客眼球，清晰传达“内网、安全、AI算力”三大核心标签。

- 主标题（H1）：类脑大模型智能体盒子 —— 您的内网私有化 AI 算力底座
- 副标题（H2）：专为政企及涉密办公场景打造，物理隔离、数据零泄露、本地毫秒级推理。彻底告别云端依赖，让数据不出工位，打造安全、高效、自主可控的智能办公新体验。
- 行动号召按钮（CTA）：[预约产品演示] [加入脑启社区]

【配图建议】：
主视觉：极具科技感的深色背景。画面中央放置一台精致小巧的“类脑盒子”（体现 25×18×10cm 的紧凑感），盒子散发微光，周围环绕着代表“安全盾牌”、“数据闭环”和“AI 神经网络”的抽象光效线条。

# 模块二：直击痛点（Pain Points） - 为什么选择我们？

【设计目标】 针对政企客户“想用大模型但不敢用、用不了”的现状，提出针对性解法。

版块标题：打破内网办公 AI 落地的三大枷锁
内容矩阵（三列布局）：
数据泄露风险高？
传统痛点：公有云 AI 需联网上传，涉密公文流转存在先天泄露风险。
我们的解法：物理级断网，纯离线推理。 移除外网通信模块，数据百分百留存本地，从源头切断泄露路径。
算力不足且依赖网络？
传统痛点：断网即停用，普通办公 PC 无法支撑大模型运行。
我们的解法：独立算力，毫秒级响应。 专属类脑 NPU，不占用主机资源，无网环境依然流畅运行。
内网数据成信息孤岛？
传统痛点：内网文档分散，无法智能检索，单点部署无法共享能力。
我们的解法：本地 RAG 引擎，局域网协同。 跨格式语义检索，打破部门壁垒，实现知识与能力全域共享。
【配图建议】：采用对比图（Before/After）或图标+卡片的形式。左侧用灰色调表现传统痛点（如断网图标、数据泄露警告），右侧用亮色调表现金山盒子的解决方案（如安全锁、极速闪电、互联节点）。

# 模块三：硬核底座（Hardware Specs） - 桌面级算力革命

【设计目标】 突出硬件的轻量化、高性能与极致安全性。

版块标题：极速、极简、极安全的桌面级 AI 终端
核心参数（四宫格展示）：
⚡ 160 TOPS 澎湃算力：搭载高性能类脑 NPU，流畅支撑 7B-13B 规模大模型边缘端运行。
🔋 <25W 超低功耗：12V低压供电，无风扇静音设计。无需改造机房与空调，普通工位即插即用。
🛡️ 全链路国密安全：内置国密 TCM 安全模块，SM4/SM2 加密标准，2TB 高速 SSD 全盘加密。
🔌 物理级安全隔离：仅保留千兆内网接口，物理移除无线网卡与外网通道。
【配图建议】：
采用“硬件爆炸图（Exploded View）”或“透视图”。将盒子拆解，清晰展示内部的类脑芯片、加密硬盘、无风扇散热结构和丰富的工业总线接口，配以科技感的数据标注线。

# 模块四：四大智能体（AI Agents） - 核心办公场景

【设计目标】 展示 SaaS 层的开箱即用能力，让客户看到实际业务价值。

版块标题：四大轻量化办公智能体，重构内网工作流
内容轮播或标签页（Tabs）：
📝 智能文档处理：支持 PDF、扫描件等异构数据。分钟级完成摘要生成、关键词提取与多语言翻译。
🧠 企业专属知识库：基于本地离线 RAG 系统，导入规章制度与技术文档，打造全天候在线的内部智能专家。
🎙️ 会议纪要与任务提取：音视频转文字，智能提炼核心观点与待办事项，推动任务高效分配。
🔒 涉密公文合规校验：针对极高安全场景，提供涉密档案结构化处理与涉密术语合规校验。
【配图建议】：
每个 Tab 搭配一张真实的 UI 界面 Mockup（如 WPS 风格的文档处理界面、类似对话框的知识库问答界面），界面需置于一台内网办公电脑的屏幕中，旁边放着类脑盒子。

# 模块五：技术架构（Architecture） - 全栈自主可控

【设计目标】 展现技术深度，增强 IT 决策者（CIO/CTO）的信任感。

版块标题：软硬深度协同，构筑四层安全办公架构
文案：打通从数据采集到算力输出的完整链路，兼顾单点安全与全域协同。
SaaS 场景应用层：公文撰写 / 知识检索 / 纪要生成
MaaS 模型服务层：轻量化 MoE 混合专家大模型 / 推测解码加速引擎
PaaS 平台调度层：本地模型微调 / 集群管理 / RAG 知识库
IaaS 硬件基座层：OpenClaw 类脑盒子 / TCM 安全模块 / 物理隔离
【配图建议】：
一张精美的 3D 架构层级图（Layered Architecture Diagram），自下而上展示 IaaS -> PaaS -> MaaS -> SaaS，辅以动态的数据流向箭头，体现“异构推理加速”和“本地闭环”

【设计目标】 展现产品的泛用性，不仅是办公，更是边缘计算大脑。

行业卡片：

🏭 智能制造：化身“产线鹰眼”，完成工业数据实时分析，毫秒级异常预警。

【配图建议】：
三张行业场景的高清实景图（政务大厅、自动化产线、监控中心），图片上叠加类脑盒子的半透明图标和 AI 识别框特效。

# 模块七：产品演进路径（Roadmap） - 陪伴企业共同成长

【设计目标】 展现产品的生命力与未来规划，引导客户从单机向集群升级。

版块标题：阶梯式能力升级，适配不同部署阶段
时间轴/阶梯设计：
V1.0 单机基础办公版：单点涉密工位首选。脱网独立运行，满足基础 AI 办公与合规校验。
V2.0 内网协同集群版：部门级部署。多设备状态监控，知识库实时同步，跨工位分布式检索。
V3.0 自主进化智能体版：企业级生态。多智能体协作，本地模型微调蒸馏，业务能力持续迭代。
【配图建议】：
阶梯向上的路线图（Roadmap）。V1 是一台盒子；V2 是多台盒子通过网络线连接；V3 是盒子上方出现复杂的神经网络和多个 Agent 协作的图标。
