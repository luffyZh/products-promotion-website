隐卫：隐私防护防泄密监控设备
【核心Slogan】一探设备，二识行为；双模融合，精准告警
一、产品定义
“隐卫”是一款专为高密级场所及核心涉密人员打造的全场景防泄密监控系统。
系统首创**“DVS事件相机 + IR红外传感”多模态融合架构，彻底颠覆传统依赖无线信号的检测手段。通过“设备探测+行为识别”双重锁定，不仅能精准抓取0.5m-8m范围内的偷拍行为，更从物理底层确保了监控设备自身“零泄密”风险**，为政企核心机密筑起最后一道防线。
- 🛡️ 【不怕断网，无惧屏蔽】—— 突破传统信号检测盲区 不依赖WiFi、蓝牙或蜂窝网络信号。无论偷拍设备处于“飞行模式”、未插卡状态，还是在开启信号屏蔽仪的绝密会议室中，依然精准识别。
- 👁️ 【设备+行为 双重锁定】—— 极速精准打击 红外(IR)传感器敏锐捕捉手机镜头等光学设备特征；DVS事件相机毫秒级感知“举机、对准、按下快门”等动态偷拍行为。融合算法综合研判，将误报率降至极低。
- 🔒 【监控者不泄密】—— 物理级隐私安全保障 传统RGB防拍摄像头存在被黑客攻破、反向拍走涉密文件的致命风险。隐卫采用的DVS技术仅输出动态像素变化（轮廓与动作），不记录高清静态图像与文字细节，从硬件源头杜绝了“二次泄密”的可能。

二、产品形态
根据不同涉密等级与应用场景，“隐卫”提供两大系列产品形态，实现从“固定空间”到“移动办公”的全方位防护：
1. 隐卫·穹顶 (Pro版) —— 固定式空间防卫
- 部署形态： 房间顶部/墙壁隐蔽式固定安装。
- 防护范围： 2米 ~ 6米中远距离广角覆盖。
- 核心场景： 党政军保密会议室、企业高管办公室、研发中心、指挥调度中心。
- 产品特性： 全天候无死角监控，支持多台设备组网联动，形成全屋防偷拍穹顶网络，精准定位异常行为坐标。
2. 隐卫·随行 (Air版) —— 便携夹戴式贴身护卫
- 部署形态： 迷你夹戴式设计（可夹于屏幕上方、文件夹或随身放置）。
- 防护范围： 0.5米 ~ 2米近场精准防护。
- 核心场景： 机密电脑操作位、外出涉密会议、临时保密审查、差旅移动办公。
- 产品特性： 即插即用/超长待机，灵活机动。专防“背后偷拍”与“越肩窥视”，让涉密人员随时随地自带“安全结界”。

三、应用场景
1. 涉密办公室
适用于保密单位、军工企业、科研院所、核心研发部门等涉密办公空间，防范内部人员违规携带手机、录音笔等设备进入并实施偷拍、录音等行为。
典型风险：
- 手机带入涉密办公区
- 拍摄电脑屏幕、文件资料、研发图纸
- 使用录音笔、移动硬盘等违规电子设备
2. 保密会议室
适用于涉密会议、项目评审会、专家论证会、招投标封闭评标区等场景，防止会议内容、汇报材料、投影屏幕被违规拍摄或录音。
典型风险：
- 参会人员举起手机拍摄屏幕
- 桌面下隐蔽录音
- 会议资料被快速翻拍
- 外来人员携带违规设备进入
3. 核心研发中心
适用于芯片、军工、能源、通信、智能制造等核心研发区域，防止技术图纸、算法代码、实验数据、样机信息等敏感资料被拍摄外传。
典型风险：
- 手机偷拍研发屏幕
- 拍摄实验设备或样机
- 非授权移动存储设备接入
- 研发成果被内部泄露
4. 机房、档案室与高密级空间
适用于核心机房、涉密档案室、资料室、保密库房等区域，对进入人员的电子设备携带与可疑操作行为进行持续感知。
典型风险：
- 拍摄服务器配置、网络拓扑、档案资料
- 使用手机、录音笔等违规电子设备
- 私自拍摄、复制或记录涉密内容
四、痛点问题与解决方案
痛点一：传统门禁和安检只能管入口，无法监管过程行为
问题描述
门禁系统解决的是“谁进入”，安检设备解决的是“进门时是否携带物品”，但人员进入涉密空间后是否掏出手机、是否对准屏幕、是否偷拍资料，传统手段难以及时发现。
解决方案
系统在涉密场所内部部署感知设备，持续识别人员在空间内的可疑行为，实现从 入口管控 到 过程行为感知 的升级。
对应能力
- 红外传感器发现手机等潜在偷拍设备
- DVS事件相机识别掏出、举起、对准、拍摄等行为
- 系统根据设备信号与行为信号进行风险判断
痛点二：普通高清监控可能采集涉密内容，存在二次泄密风险
问题描述
传统摄像头通过高清画面进行监管，但在涉密场所中，高清画面可能拍到屏幕内容、文件资料、人脸信息或会议材料，监控本身可能成为新的泄密源。
解决方案
系统不以“看清内容”为目标，而是通过 红外热源特征 与 DVS事件变化特征 识别风险行为，减少对涉密内容的采集依赖。
对应能力
- 红外传感器识别设备热源，不读取屏幕内容
- DVS事件相机关注动态变化，不依赖高清连续图像
- 告警信息可脱敏展示
- 数据处理可在内网封闭环境完成
痛点三：单一传感器容易误报或漏报
问题描述
如果只依赖视觉动作识别，可能把喝水、拿笔记本、伸手等动作误判为偷拍；如果只依赖红外探测，可能把热水杯、暖风、其他热源误判为手机。
解决方案
系统采用 红外传感器 + DVS事件相机 的多模态融合方案，将“设备存在”与“行为动作”进行交叉验证，降低单一感知带来的误报和漏报。

涉密空间对数据传输、数据存储、网络边界和设备形态有严格要求，系统既要具备智能识别能力，又不能增加新的安全风险。

五、核心技术优势
红外传感器：探测有没有偷拍设备？
红外传感器用于感知手机、录音笔、移动硬盘等电子设备的热源特征，识别潜在偷拍或违规电子设备。
- 可识别手机等电子设备的异常热源
- 可辅助发现藏匿于口袋、衣物、桌面附近的设备
- 不读取屏幕内容，不采集文字细节
- 与DVS行为识别形成互补
DVS事件相机：感知有没有偷拍行为？
DVS事件相机只响应场景中的动态变化，适合捕捉掏出手机、举起手机、对准屏幕、短暂拍摄等行为轨迹。
- 低延迟，适合捕捉快速动作
- 低数据量，适合内网部署
- 对静态屏幕内容不敏感
- 适合识别微小、隐蔽、突发行为
多模态融合算法：是否构成真实风险？
系统将红外探测到的设备信号与DVS感知到的行为信号进行时间同步、空间对齐和置信度融合，输出风险等级。
- 设备信号与行为信号交叉验证
- 降低误报和漏报
- 支持风险分级
- 支持多场景参数调优
- 支持扩展识别录音笔、移动存储设备等违规设备
多形态部署，适配不同场景
系统可根据场所安全等级、部署规模、隐蔽性要求和预算条件，选择不同部署架构以及产品形态，最终实现 0.5m 到 6米范围内的全方位隐私防护监控。

六、功能指标
- 识别场景： 违规携带手机、隐蔽偷拍行为、越肩窥视拍摄。
- 极速响应： ≤ 3秒 （从偷拍行为发生，到终端/后台接收告警，毫秒级抓拍，秒级响应）。
- 探测纵深： 0.5米 ~ 6米 （随行版0.5-2m，穹顶版2-8m，支持多点组网）。
- 识别精度： ≥ 80% （复杂光线与遮挡环境下依然保持高准度）。
- 抗干扰性： 误报率 ≤ 10% （精准过滤喝水、挠头、正常翻阅文件等日常动作）。

七、竞品方案/传统方案对比
从“信号检测”到“设备+行为”融合判断
传统 WiFi 检测主要依赖无线信号，遇到飞行模式、无信号设备或信号屏蔽环境时容易失效；普通RGB防偷拍设备虽然能识别动作，但自身存在采集涉密内容的风险；夹带式屏幕防偷拍设备部署灵活，但覆盖距离和场景有限。

八、营销优化建议：
1. 强化“矛与盾”的故事： 在宣传时，可以强调现在很多商业间谍/窃密者会把手机开飞行模式带入会议室录像，传统设备（盾）已经失效，而“隐卫”是新一代的盾。
2. 视觉呈现建议： 在画册或PPT中，一定要放一张**“DVS视觉效果图” vs “普通摄像头效果图”**的对比。让客户直观看到DVS画面只有人体的边缘轮廓线，看不清桌上的机密文件字迹，这比一万句解释都管用。
3. 主打“组合拳”： 销售策略上，可以推行**“1个会议室穹顶 + N个随行版”**的套装模式，提升客单价。
